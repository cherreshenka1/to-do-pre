# Сервис To-do

Интерактивное приложение для управления списком дел с сохранением данных в локальное хранилище.

## Описание проекта

«Сервис To-do» — это одностраничное приложение, которое позволяет пользователю:

- Просматривать список текущих задач
- Добавлять новые задачи
- Удалять задачи
- Копировать существующие задачи
- Редактировать текст задачи
- Сохранять все изменения в локальном хранилище браузера

Проект выполнен в рамках учебной программы и представляет собой **написание JavaScript-логики** к уже готовой HTML-разметке и CSS-стилям.

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
  - Работа с DOM (querySelector, createElement, cloneNode)
  - Обработка событий (addEventListener)
  - Работа с localStorage
  - JSON.parse / JSON.stringify
  - Методы массивов (forEach)
- Git / GitHub

## Функциональность

Приложение обладает следующим функционалом:

1. **Отображение задач** — при загрузке страницы показываются сохранённые задачи или начальный список
2. **Добавление задач** — через форму ввода новая задача добавляется в начало списка
3. **Удаление задач** — по нажатию на кнопку с корзиной
4. **Копирование задач** — по нажатию на кнопку копирования задача дублируется в начало списка
5. **Редактирование задач** — по нажатию на кнопку с карандашом текст становится редактируемым
6. **Сохранение данных** — все изменения автоматически сохраняются в localStorage
